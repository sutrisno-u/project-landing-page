# Project Landing Page - The Odin Project

This project is a part of **The Odin Project** curriculum. The main focus of this assignment is to practice **Flexbox** and implement **Semantic HTML**.

[🚀 Live Demo Here](https://sutrisno-u.github.io/project-landing-page/)

## ✨ Key Features
*   **Responsive Design**: The layout adapts to various screen sizes.
*   **HTML5 Semantic Structure**: Clean code using tags like `<header>`, `<section>`, and `<footer>`.
*   **Flexbox Layout**: Precise element positioning using modern CSS Flexbox.

## 🛠️ Tech Stack
*   **HTML5** (Semantics)
*   **CSS3** (Flexbox)

## 💡 What I Learned
During this project, I deepened my understanding of how to align and distribute space between elements using `flex-direction`, `justify-content`, and `align-items`. It was a great exercise in building a clean and functional landing page from scratch.
